Activated Scene

An activated scene is the scene currently defined as the working/load in scene, setting a scene as the Activated scene moves the actors to the newlyl activated scene.  You activate a scene in either the Scene Navigator Bar, and the Sidebar Tab (Scene).

Actor

Actors is the term for a character sheet's subject and the sheet itself.  Creatures; PCs, NPCs, Monsters, etc -- are all types of Actors.  Actors is used as a generic term for any RPG's character.

Add-On Module

(Module for short)

In Foundry, Add-On modules serve as packages that can; build out new features; enhance existing features; add resources (often as associated Compendia).
You add modules via the Configuration &Setup screen's section either using Manifest URLS; or through the dialog windows. You can see a listing of offical modules here. Some developers, like Grape Juice, may not have their work listed on such, so you might need to keep an eye out for mentions of ones that are "off the books".

Adventure(s)

Small parts of a campaigns/Adventures have been traditionally called Modules (eg the Tomb of Horros, was the module S1). Adventures in Foundry however are part of structure called a World.

 

Alpha

Alpha is part of the product lifecycle. It is a pre-release being submited for dedicated testing.  Using Alpha tends to imply that the user is agreeing to document and test, isolate, and report issues with that alpha release.

Ambient Light Source

(Light Source)
Ambient light sources are placed on the Canvas Lighting Layer.Ambient Light sources are configured to emit light (releave a portion of the Battlemap). Each Light Source has a number of other controls for its coloration, "strength" (how much of map it exposes), and at what light level they turn on/off.

Ambient Sound

Ambient sound are placable items on the Canvas Soudnds. Ambient sounds have an AoE and can have attenuation (the volume drops as you further away from the placable epicenter) or a region defined in which if someone enters the sound is constantly there, and consistantly loud.

 

Ambient Sound Controls

Ambient Sound Controls are part of the Canvas Controls that everyone has access to. They are used for viewing/interacting with pinned notes on the current scene and consist of: Draw Ambient Sound, Clear Sound.

 

AoE

(Area of Effect)
is used to determine who were effected by the effective area. Foundry provides several preconfigred template to play to help identify the areas of effect.

Apache

Is  a free, open source, cross platform server that runs about 40% of the websites on the internet. Its very common for hosting companies and Linux users to use/support Apache. 
For configuration information see Foundry's Apache configuration page.

API

(stands for Application Programming Interface)
There will be a lot of mention APIs in Foundry documentation. This is largely because Foundry was written very much around being able to be expanded upon.  Its not really relevant beyond noting that this is why you'll see so many Add-On Modules, and why they vary so greatly on what they can do, so if you have some talent or a desire to develop some skills in programming, most likely you can use the APIs to do what it is envision.  (talk with the discord server's #module-development channel for help building such)

APP

(also called "the app" or Electron)
GM Guide to the Interface provides an overview of such. This is an interface built from a stripped down version of a Chrome's development tool browser which they call Electron.  When people refer to opening "The App" they are most often talking about the launching of Electron browser into the Setup and Configuration. It is possible to run or even play from the APP, however; its not advised.

Application Data

The directory into which the Foundry application has been installed.  This becomes relevant if you're experiencing problems and need to address licensing or check/tweak the system.  In windows you can find it by right mouse clicking on the open App (electron) and choosing Browse Application Data.

Audio

Background

The term for the image defined in the configuration of a Scene. It is the furthest back image possible and it is not selectable, and so it cannot be accidently grabbed, resized or deleted when attempting move tokens or tiles.

Beta

Is the phase in software development between ALPHA and stable/release candidate. Beta software is generally considered "complete" by the developer but still in need of a larger testing cycle to ensure bugs were not introduced with the new features/bug fixes..

Blind Roll

is a Chat Command.  A "blind roll" is a private dice roll which where the result and details of the roll are only visible to Game Master users. The rolling player will not see the result of their own roll. This is similar to using a dice tower or other device at a physical tabletop where the roller may not get to see the outcome. To perform a blind roll, use  /blindroll, /broll, or /br as the command prefix.

Compendium

(Plural Compendia)

this is the hardest to describe easily - a compendium is a set of a type of content (a grouping of Actor, Scenes, Rollable Tables, Items, Macros, Playlists). These are stored outside of the game proper which allows access to the information but doesn't require the system to hold the information in memory.  They however don't have the ability to sort them into folders. Door

For foundry a Door is a type of Wall which when used, produces a special icon on the scene that players can see as well. Doors are the only wall types have the possibility of being interacted with by players - they have the ability to "open" allowing actor's tokens to see and move beyond their previous solid boundry. Doors can be Locked Doors, which will not open when clicked, but do denote they're a door via a visual icon.  
Secret Doors differ in that they cannot be seen or opened without GM/or scripted triggers.

Drawing Tool(s)

is a type of Canvas Control. There are 5 Canvas Control's Tools currently under it; Rectangle; Ellipse; Polygon; Freehand and Text drawing tools. These are used to create shapes on the canvas.
By default players cannot see or use the drawing tools unless they're a Trusted Player. The GameMaster however can grant access through the Game Settings menu's Permission Configuration, specifically the "use Drawing Tools" option.

Reverse Proxy

A reverse proxy takes requests from the internet clients and directs them to specific resources, obfucating their location.  Its like a Proxy, but with the the focus on dealing with incoming requests and allow your Foundry Server to be secured behind that service, including the ability to change your machine's IP.
For Foundry is is often used to bypass issues with ISPs restrictions allow you to have a server/service that detects requests to a URL which it will then forward your Foundry server's IP, even if that IP changes frequently or your provider doesn't allow establishing DMZs/Port Forwarding from your router.
Some common servers for this are Nginx; Apache; and Caddy;

This is a special kind of Wall. that allows for one way vision or movement.  Terrain Walls show Green on the canvas's wall layer.

Tile

In Foundry any graphic that isn't a Actor token on the battlemap (Canvas/Scene) is a tile.  Tiles are displayed above the background image of the scene. Tiles add decorations, props, or custom features. Tiles do not need to be Geomorphic (built to fit together to make a larger map), nor do they need to be a map element - functionally a tile is a graphic image that cannot be manipulated by the players.
